
# 🌲 Forest Drag and Drop Web Application

An interactive forest-themed web page where users can drag and drop animals into a forest scene.
This project is built using **HTML, CSS, and JavaScript** in a single file structure.

---

## 🚀 Features

* 🌳 Forest background interface
* 🐯 Drag and drop animals into the forest
* 🎯 Interactive UI experience
* 🖼️ Uses custom animal images
* ⚡ JavaScript-based drag and drop functionality

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
Forest-Drag-Drop/
│
├── forest.html   (Contains HTML, CSS, and JS in one file)
└── images/
    ├── lion.png
    ├── elephant.png
    ├── tiger.png
    └── ...
```

---

## ▶️ How to Run

1. Download or clone the repository:

   ```
   git clone https://github.com/your-username/Forest-Drag-Drop.git
   ```
2. Open the folder.
3. Double-click `forest.html`.
4. Start dragging animals into the forest 🌲🐘
